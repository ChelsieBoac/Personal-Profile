# Personal Profile Website

## Overview

This project is an upgraded personal profile website developed as part of our Midterm Activity. It showcases my personal information, skills, projects, and contact details using modern web technologies. The website was enhanced from the original version by adding interactive features, improving the user interface, and integrating a cloud database.

## Features

* Responsive personal portfolio website
* Navigation bar with smooth scrolling
* About Me section
* Skills section with progress bars
* Projects section
* Contact form
* Live date and time
* Dark Mode toggle
* Scroll-to-top button
* Firebase Firestore integration for storing contact form submissions
* Hosted using GitHub Pages

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Firebase Firestore
* Git
* GitHub
* Visual Studio Code
* ChatGPT (AI Assistant)

## AI Tools Used

ChatGPT was used to assist in designing the user interface, generating HTML, CSS, JavaScript code, improving the website layout, and integrating Firebase Firestore.

## Version Control

This project uses Git and GitHub for version control. A separate branch was created to implement improvements before merging them into the main branch.
