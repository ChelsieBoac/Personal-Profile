# Personal Profile Website

## Project Description

This project is an AI-powered personal profile website created using HTML, CSS, and JavaScript. It presents my personal information, educational background, skills, hobbies, and contact details in a simple and professional design.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub Pages
* Visual Studio Code

## AI Tools Used

* ChatGPT – for generating content and coding assistance.
* GitHub Copilot – for coding suggestions.

## Features

* Personal profile information
* About Me section
* Skills section
* Educational background
* Contact information

## Challenges Encountered

During the development of this project, I experienced difficulties in setting up Git and connecting my local repository to GitHub. I also encountered errors when pushing my files and publishing the website. These problems were resolved by configuring Git correctly and following the proper Git commands.

## Improvements Made

I improved the layout by adding colors and organizing the content into sections to make the website more visually appealing and easier to read.
